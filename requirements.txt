## 🌟 Project Overview
The **AI Email Classification & Auto-Reply System** is an intelligent triage tool designed to solve the problem of "inbox overload." By using Natural Language Processing (NLP) and rule-based AI, it automatically analyzes the intent and urgency of incoming emails, categorizes them, and suggests a professional response instantly.

### 🎯 Why Use This System?
- **⚡ Instant Triage:** Automatically identifies critical "Support" issues so they don't get lost.
- **🕒 Efficiency:** Saves hours of manual sorting for customer service and sales teams.
- **✅ Consistency:** Applies the same professional logic to every email, reducing human error.
- **📈 Scalability:** Helps growing businesses handle high email volumes without increasing headcount.

## 🛠️ How It Works
1. **Preprocessing:** The system cleans the input by removing punctuation and converting text to lowercase to ensure consistency.
2. **Category Classification:** The AI scans for specific "trigger phrases" (e.g., "not arrived" triggers Support, "price" triggers Sales).
3. **Urgency Detection:** It evaluates the combination of keywords and categories to assign a Priority (e.g., any "Support" message is flagged as High Priority).
4. **Smart Reply:** A tailored response template is selected based on the final classification.

## How to Run
1. Ensure you have Python installed.
2. (Optional) Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the main script:
   ```bash
   python main.py
   ```
4. Enter the email text when prompted. Type `exit` to quit.
